# Lottie 애니메이션 파일

이 폴더에 동물 Lottie JSON 파일을 추가하면 자동으로 농장에 표시됩니다.

## 파일 이름 규칙

| 파일 | 동물 |
|------|------|
| `cat.json` | 고양이 |
| `dog.json` | 강아지 |
| `rabbit.json` | 토끼 |

## 파일 없을 때

JSON 파일이 없으면 자동으로 이모지 애니메이션으로 대체됩니다.

## 추천 소스

- https://lottiefiles.com 에서 검색 (cat farm, dog cute, rabbit animation 등)
